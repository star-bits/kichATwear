# kich@wear

- 외부온도는 tag expression으로 넣을 수 없음. 
- 따라서 위젯으로 넣게 하되, 온도 텍스트만 출력하게 설정하고, 텍스트의 위치, 폰트, 크기, 색깔을 지정해주었음. 
- 갤럭시 워치의 기본 날씨 위젯으로 설정하면 아래와 같이 표시됨. 온도가 한 자리 수인 경우, 영하인 경우, 두 자리 수인 경우의 텍스트 위치를 확인함. 
- 갤럭시 워치 5 40mm 모델에서 확인함.
